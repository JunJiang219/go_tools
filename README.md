# go_tools
基于golang开发的命令行工具

## 目录结构
```
mycli/                  # 项目根目录
├── cmd/                # 【核心】存放所有子命令
│   ├── root.go         # 根命令定义
│   ├── add.go          # 例如：`mycli add` 子命令
│   └── list.go         # 例如：`mycli list` 子命令
├── internal/           # 【核心】私有应用程序和库代码（禁止外部导入）
│   └── app/           # （可选）内部应用逻辑
├── pkg/               # 【核心】公共库代码（允许外部项目导入）
│   └── utils/         # 例如：公共工具函数
├── main.go            # 【核心】程序主入口，通常只初始化并执行根命令
├── go.mod             # Go模块定义文件
├── go.sum             # 模块依赖校验文件
└── configs/           # （可选）配置文件模板
```
